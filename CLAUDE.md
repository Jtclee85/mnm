# CLAUDE.md

## 프로젝트 요약

이 프로젝트는 Next.js 기반 교육용 웹앱 「뭐냐면」입니다.

「뭐냐면」은 초등 사회과 조사학습과 박물관 학습에서 학생들이 접하는 어려운 조사자료, 전시 설명문, 안내문 등을 AI로 학생 수준에 맞게 재구성해주는 웹앱입니다.

이 앱은 단순 AI 챗봇이 아니라, 학생이 조사자료를 이해하고, 탐구 질문을 만들고, 발표로 표현하고, 자신의 학습을 성찰하도록 돕는 "깊이있는 학습 지원 AI 코스웨어"를 목표로 합니다.

상세한 개발 원칙과 하네스 규칙은 `@HARNESS.md`를 따릅니다. 작업 전 반드시 `HARNESS.md`를 참고하세요.

## 핵심 작업 원칙

- AI는 학생 대신 학습하지 않는다.
- AI는 학생이 생각하도록 돕는다.
- 새 기능은 이해·탐구·표현·성찰 중 하나 이상을 지원해야 한다.
- 기존 기능을 이유 없이 제거하지 않는다.
- 기존 UI를 사용자 동의 없이 크게 바꾸지 않는다.
- 큰 구조 변경 전에는 반드시 계획을 설명하고 확인을 요청한다.
- 코드 수정 후 가능한 테스트를 실행한다.
- 테스트 통과 전에는 commit/push를 제안하지 않는다.
- API 키나 민감정보를 코드에 직접 넣지 않는다.

## 작업 절차

1. 작업 목표 재진술
2. 구현 계획 설명
3. 영향받는 파일 목록 설명
4. 코드 수정
5. 테스트 실행 또는 테스트 불가 사유 설명
6. 변경사항 요약
7. commit/push 제안
