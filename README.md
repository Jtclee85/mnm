# GPT 챗봇 (Simple Template)

## 🚀 시작 방법

1. `.env.local.example` 파일을 `.env.local`로 복사하고 OpenAI 키 입력
2. 로컬에서:
   ```
   npm install
   npm run dev
   ```
3. 브라우저에서 `http://localhost:3000` 접속 → 챗봇 테스트

## ☁️ Vercel 배포

1. 저장소를 Vercel에 연결
2. 환경변수 `OPENAI_API_KEY` 등록
3. Deploy → 생성된 URL로 접속하면 ✅ 바로 챗봇 사용 가능
