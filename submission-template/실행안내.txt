[오프라인 실행 방법]

1. USB의 program 폴더를 연다.
2. index.html을 더블클릭한다.
3. 시작 화면에서 오프라인 시연 보기를 누른다.
4. 별도 서버 없이 Chrome, Edge, Safari에서 실행할 수 있다.
5. AI 실시간 응답은 온라인 배포판에서 작동하며, 오프라인 시연은 저장된 예시 데이터로 실행된다.

[수동 검수 방법]

1. program/index.html을 더블클릭한다.
2. 오프라인 시연 보기를 눌러 program/offline-demo/index.html이 열리는지 확인한다.
3. 실제 앱 UI와 챗봇 버튼이 보이는지 확인한다.
4. 학습산출물 공유하기를 눌러 program/share/index.html?d=... 새 창이 열리는지 확인한다.
5. 공유 페이지에 저장된 예시 산출물이 표시되는지 확인한다.
6. 개발자도구 Console과 Network에서 _next 리소스 404가 없는지 확인한다.

[추천 영상(함께 보면 좋은 영상) 저작권 안내]

본 소프트웨어의 '함께 보면 좋은 영상' 기능은 YouTube 영상 및 썸네일 이미지를
프로그램 내부에 저장·복제·재배포하지 않고, 교사가 사전에 승인한 공공·교육기관
공식 YouTube 채널의 원본 영상 정보를 YouTube 공식 API 또는 임베드 기능을 통해
연결하는 방식으로 구현하였다. 영상 제목, 채널명, 출처를 화면에 명시하였으며,
동영상 파일·음원·자막·썸네일 이미지 파일을 제출물에 포함하지 않았다.
오프라인 실행 환경에서는 YouTube API를 호출하지 않고, 저장된 예시의 videoId와
원본 링크를 통해 외부 참고자료임을 표시한다.
